# 线条操作

## 属性

### lineWidth 线宽

```js
ctx.lineWidth = number;
```

### lineCap 线帽

```js
// round 圆角
// square 正方形
// butt 默认
ctx.lineCap = "属性值";
```

### lineJoin 交叉处样式

```js
// miter 尖角
// round 圆角
// bevel 斜角
ctx.lineJoin = "属性值";
```

## 方法

### setLineDash 虚实样式

```js
// array [10,5] 10px实现 5px空白
ctx.setLineDash(array);
```
